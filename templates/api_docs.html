GET /api/v1/positions

Liston te gjitha pozicionet e hapura me informacione per kompaninë dhe profesionin.

GET /api/v1/stats

Statistika te platformes: numri i profesioneve, pozicioneve, kompanive, nxenesve, kontratave, certifikatave.

GET /api/v1/docs

Dokumentacioni i API-se ne format JSON.

Integrimi

Kjo API eshte e hapur per t'u perdorur nga shkollat, kompanite, dhe zhvilluesit. Nuk kerkohet autentikim per endpointet publike.

Per akses te avancuar (menaxhim aplikimesh, kontratash), kontaktoni administratorin per API key.
